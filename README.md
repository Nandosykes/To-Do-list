# Gerenciador de Tarefas (To-Do List)

Projeto desenvolvido com o objetivo de praticar **lógica de programação**, **organização de código** e **desenvolvimento front-end** utilizando tecnologias básicas da web.

## Visão Geral
O Gerenciador de Tarefas é uma aplicação simples que permite ao usuário adicionar, remover e marcar tarefas como concluídas.  
O foco do projeto é demonstrar fundamentos de programação e boas práticas iniciais de desenvolvimento.

## Funcionalidades
- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Interface simples e intuitiva

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)
